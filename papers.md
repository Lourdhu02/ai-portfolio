# Papers & Architectures

1. **PAHF: Personalization Agent with Hybrid Framework** (Meta, Feb 2026)
   https://arxiv.org/abs/2602.16173
   Three-step agent loop (pre-action clarification → memory-grounded action → post-action feedback) for continual user personalization.

2. **Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks** (Lewis et al., 2020)
   https://arxiv.org/abs/2005.11401
   Foundational RAG paper combining parametric memory with non-parametric vector retrieval.

3. **REACT: Synergizing Reasoning and Acting in Language Models** (Yao et al., 2022)
   https://arxiv.org/abs/2210.03629
   Interleaved reasoning traces + tool calls — basis for agentic tool-use patterns in portfolio agents.

4. **Model Context Protocol (MCP) — Anthropic**
   https://modelcontextprotocol.io
   Standardized tool protocol for LLM-host communication; used by casper-ai-portfolio-agent and financial agents.

5. **Vercel AI SDK RAG Guide**
   https://ai-sdk.dev/cookbook/guides/rag-chatbot
   Streaming RAG chatbot pattern with streamText, tool calls, and vector search — reference for production AI portfolios.
