# Model Context Protocol (MCP)

**URL:** https://modelcontextprotocol.io
**Source:** Anthropic

MCP (Model Context Protocol) is an open-source standard for connecting AI applications to external systems. Like USB-C for AI — standardized way to connect AI apps to data sources, tools, and workflows.

Key capabilities:
- Agents access Google Calendar, Notion, etc.
- Claude Code generates web apps from Figma designs
- Enterprise chatbots query multiple databases
- AI models create 3D designs on Blender

Supported by: Claude, ChatGPT, VS Code, Cursor, MCPJam
